LA LIGNE DE COMMANDE

Gestion de documents

Commandes à retenir

  • Afficher le contenu d'un dossier >> ls
  • Afficher le contenu d'un dossier avec les fichiers cachés >> ls -A
  • Afficher le contenu d'un dossier sans les fichiers cachés >> ls - l
  • Afficher le contenu d'un dossier version longue avec les fichiers cachés ls -Al
  • Afficher le chemin d'un dossier >> PWD
  • Afficher le contenu du dossier "Documents" (exemple) >> ls/home/francky/Documents

  • Se mettre en super utilisateur >> sudo -s + mode de passe
  • Mettre à jour la liste de paquets >> sudo apt update
  • Mettre à jour les paquets >> sudo apt upgrade
  • Trouver tous les paquets contenant le mot "XML" (exemple) >> sudo apt search XML
  • Installer le paquet "git" (exemple) >> sudo apt install git
  • Lancer une installation d'un fichier téléchargé >> sudo dpkg -i + nom du dossier téléchargé

  • Créer un dossier "website" dans le dossier "Documents" >> cd /home/francky/Documents mkdir website
  • Créer un dossier "Memento" (exemple) et un sous-dossier "assets" en une seule ligne de commande dans le dossier website >> mkdir -p Memento/assets
  • Créer 3 dossiers "css" "img" "js" en une seule ligne de commande >> mkdir css img js
  • Créer un fichier "stylee.css" (exemple) >> touch style.css
  • Déplacer le fichier "style.css" dans le dossier "css" du dossier "assets" >> mv style.css /Documents/website/Memento/assets/CSS
  • Ouvrir un fichier avec nano >> nano index.html
  • Ouvrir un fichier avec le navigateur internet par défaut >> x-www-browser index.html ou firefox index.html si on le connait déjà
  • Ouvrir les options nano >> CTRL + ALT GR + 8

  • Faire une liste >> ouvrir avec < UL > puis < li > avec le contenu puis fermer par < /li > et fermer par < /UL >

Création d'un dossier sur le Terminal + envoi sur Github

  • mkdir "Nom du dossier"
  • Aller dans le dossier
  • git init
  • touch index.html (création du fichier html)
  • Aller sur Github
  • New Repository "Nom du dossier"
  • git remote add origin + URL du repo (une seule fois/repo)
  • git add index.html
  • git commit -m "Ajout d'index"

Si ajout ou modification de dossier

  • Aller sur Atom : atom index.html et modifier le fichier (exemple : j'ajoute un message de Bienvenue)
  • git add index.html
  • git commit -m "Ajout d'un message de bienvenue"
  • git push -u origin master (envoi du dossier sur Github)
